# IIMT University Website

## Overview
This is a modern, responsive website for IIMT University featuring an elegant design with animations, smooth scrolling, and interactive elements.

## Features

### Design Elements
- **Modern UI**: Clean and professional design with gold and red color scheme
- **Animated Background**: Floating particles and geometric shapes for visual appeal
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Entrance animations for cards and interactive hover effects
- **Navigation**: Fixed navbar with smooth scroll navigation and mobile menu support

### Sections

1. **Hero Section**
   - University branding with animated title
   - Call-to-action button directing to main university website
   - Animated particles and geometric shapes

2. **Features Section**
   - Grid layout showcasing 6 key programs:
     - Engineering Excellence
     - Business Innovation
     - Pharmaceutical Sciences
     - Legal Mastery
     - Research Excellence
     - Global Exposure

3. **Stats Section**
   - Key statistics with animated counters:
     - 30K+ Students Empowered
     - 800+ Expert Faculty
     - 98% Placement Success
     - 120+ Programs Offered

4. **Footer**
   - Contact information
   - Academic programs links
   - Recognition badges

## Color Scheme
- **Primary Red**: #8b0000
- **Dark Red**: #660000
- **Gold**: #ffd700
- **Light Gold**: #fff3a0
- **Dark Background**: #000000

## Interactive Elements
- **Navbar**: Changes appearance on scroll with blur effect
- **Buttons**: 
  - Primary button with gradient background
  - Secondary button with golden border that turns red when clicked/visited
- **Feature Cards**: Lift up on hover with enhanced shadow
- **Stat Numbers**: Animate when scrolling into view
- **Geometric Shapes**: Parallax effect on scroll

## Technologies Used
- HTML5
- CSS3 (with animations and gradients)
- JavaScript (for interactivity and animations)
- Google Fonts (Poppins)

## Browser Compatibility

- Chrome
